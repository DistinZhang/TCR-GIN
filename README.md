# TCR-GIN Streamlit Demo

This branch contains the lightweight, standalone Streamlit deployment package
for the TCR-GIN web application.

The full research code, experiment scripts, Zenodo data link, and reproduction
instructions are maintained on the `main` branch.

## Streamlit Entry Point

```text
webapp/app.py
```

## Bundled Demo

The app includes a compact transport-network demo under:

```text
webapp/examples/transport_demo/
```

This package is intended for interactive web deployment only. Full datasets,
trained model collections, and result artifacts are distributed through Zenodo.
